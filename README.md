# Web Devlopment Project — CI Global Technologies Internship

Front-end web pages for an e-commerce/ordering-style application, built during a 1-month internship at CI Global Technologies, developed within a .NET framework as part of a real development team.

## Overview

A shopping/ordering web application featuring user login, item selection, cart management, and PDF receipt generation — built to simulate a real-world retail/checkout flow.

## Features

- **Login page** for user authentication
- **Home page** to browse and select items
- **Add to Cart** functionality with running cart total
- **PDF receipt generation** — prints an order summary/receipt as a downloadable PDF
- Responsive front-end layout using HTML, CSS, and JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript
- .NET framework (integration)
- PDF generation (receipt printing)

## Workflow

1. User logs in via the login page
2. Browses available items on the home page
3. Adds selected items to cart
4. Cart calculates and displays the running total
5. On checkout, a receipt is generated and printed as a PDF

## What I Learned

- Building a multi-page front-end flow (login → browse → cart → checkout)
- Implementing cart logic and dynamic total calculation with JavaScript
- Generating downloadable PDF documents from web app data
- Real-time collaboration and code review within a live development team

## Status

Built during internship — a first hands-on project applying front-end development skills to a functional e-commerce-style flow in a professional setting.
